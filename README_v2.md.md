<h2 id="english-description--英文說明">English Description / 英文說明</h2>
<p>This repository contains the final draft of a research/discussion paper concerning AI alignment through meta-dialogue frameworks. The goal is to explore conversational methods to guide AI behavior toward human values and to document a case study of a highly customized human-AI collaboration.</p>
<p>The project is open source under the MIT License. Contributions and feedback are welcome.</p>
