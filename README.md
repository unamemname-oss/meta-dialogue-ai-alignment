## English Description / 英文說明

This repository contains the initial draft of a research/discussion paper concerning AI alignment through meta-dialogue frameworks. The goal is to explore conversational methods to guide AI behavior toward human values.

The project is open source under the [MIT License](github.com). Contributions and feedback are welcome.
